# Play Store Release Notes (0.1.7+9)

## EN (Short, for Play Console)
- Improved Android alarm reliability after reboot and time/timezone changes
- Improved lock-screen alarm behavior with stronger foreground ringing flow
- Added vibration intensity options: Gentle / Standard / Aggressive
- Added settings guidance for exact alarm permission and battery optimization

## EN (Detailed, user-facing)
This update focuses on making alarms more reliable on Android devices.

- Alarms are now re-scheduled more reliably after device reboot or time changes
- Ringing flow on lock screen is more robust during idle states
- You can now choose vibration intensity per alarm (Gentle, Standard, Aggressive)
- Settings now include guidance for exact alarms and battery optimization to improve reliability
