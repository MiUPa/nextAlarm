# Play Store Release Notes (0.1.6+8)

## EN (Short, for Play Console)
- Adjusted the Add Alarm button to a smaller, Clock-style size
- Reduced the Alarm header text size to better match Android Clock styling
- Minor UI polish and stability updates

## EN (Detailed, user-facing)
This update focuses on visual consistency with the Android Clock experience.

- The floating Add Alarm (+) button has been resized to a more standard Clock-like size
- The Alarm header text has been tuned to a smaller size for a cleaner top area
- Minor UI and stability improvements
