# Play Store Release Notes (0.1.8+12)

## EN (Short, for Play Console)
- Added vibration intensity settings (Gentle / Standard / Aggressive) for alarms
- Improved Android alarm audio/vibration behavior for Bluetooth playback and Do Not Disturb scenarios

## EN (Detailed, user-facing)
This release makes WakeShock alarms harder to miss on Android.

- Added per-alarm vibration intensity control: Gentle, Standard, and Aggressive
- Improved native Android ringing flow so alarm vibration and sound are coordinated more reliably
- Better behavior when another app is playing audio over Bluetooth
- Improved handling around Do Not Disturb by using alarm-specific Android audio attributes
