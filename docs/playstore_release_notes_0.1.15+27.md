# Play Store Release Notes (0.1.15+27)

## EN (Short, for Play Console)
- Fixed Android gradual alarm volume so it now ramps up during native alarm playback
- Improved Android permission prompts for notifications, microphone, and activity access
- Improved fallback handling when challenge permissions are unavailable during ringing

## EN (Detailed, user-facing)
This release focuses on Android alarm reliability and permission guidance.

- Fixed gradual volume on Android so alarms now start quietly and increase during native ringing
- Added clearer permission explanations before Android notification, microphone, and activity prompts
- Improved denied-permission handling with better guidance to Android Settings when needed
- Improved voice and steps challenge fallback behavior when permission is unavailable during an active alarm
