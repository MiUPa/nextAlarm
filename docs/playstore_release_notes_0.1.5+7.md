# Play Store Release Notes (0.1.5+7)

## EN (Short, for Play Console)
- Added an in-app update prompt
- Improved the in-app review prompt flow
- Improved step challenge stability, including a clearer fallback when step sensors are unavailable
- Fixed minor bugs and improved overall stability

## EN (Detailed, user-facing)
In this release, we focused on making wake-up flows more reliable and easier to use.

- You'll now see an in-app prompt when an update is available
- The review prompt experience has been improved
- The step challenge is more robust, and devices without a working step sensor now have a clearer fallback path to stop alarms
- Minor bug fixes and general stability improvements
