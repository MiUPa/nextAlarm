# NextAlarm 0.1.11+23

## English
- Challenge alarms now open directly in the challenge screen without an extra start step.
- Force-closing the app while an alarm is ringing now stops playback instead of leaving it stuck on the device.

## Japanese
- チャレンジ付きアラームは、余計な開始ボタンを挟まず最初からチャレンジ画面を表示するようにしました。
- アラーム鳴動中にアプリを強制終了した場合でも、音が止まらなくなる不具合を修正しました。
